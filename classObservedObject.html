Detailed Description

template<typename T>
class ObservedObject< T >

A class that represents an object that can be observed by listeners.

The class allows registering listeners that will be notified when the object's value changes or when the object publishes changes. The class is templated on the type of the value that the object holds.

Template Parameters
TThe type of the value that the object holds.

ObserverToken ObservedObject< T >::addListener ( std::function< void(const T &)>  callback)
inline

Add a listener and return a token with a pointer to this object and a cancel lambda.

The listener is a function that takes the current value of the object as a parameter and performs some action. The token is an object that allows canceling the listener by calling its cancel() method.
