Detailed Description

A class that aggregates the data from different sources, such as the CAN bus and the sensors, and provides them as observed objects. It uses the ApplicationTypes.h file to define the types of the data, such as velocity_t and voltage_t. It also uses the DataAggregatorWrapperType.h file to wrap the data aggregator object for the C interface.
